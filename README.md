# TPI-PlanAR

# INTEGRANTES:
Ordoñez Alan y Echegaray Santiago

# DESCRIPCION:
PlanAR es una aplicación web desarrollada para la gestión y visualización de eventos. Permite a los usuarios consultar eventos disponibles, realizar reservas y visualizar distintas categorías de eventos de manera dinámica.

# TECNOLOGIAS UTILIZADAS:
HTML
CSS
JavaScript
Git
GitHub

# Funcionalidades desarrolladas:
Diseño y Estructura

Funcionalidades JavaScript

Botón "Reservar lugar".(Mensaje de confirmación mostrado dentro de la página.)
Botón "Mostrar categorías"(Visualización dinámica de categorías mediante un array JavaScript.)
Botón "Volver al inicio".
